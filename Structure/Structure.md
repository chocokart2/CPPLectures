# 구조체 (Structure)

* 구조체 : 프로그램 내에서 서로 관련이 있는 변수들을 함꼐 묶어서 새로운 자료형을 만들어 내는 방법

* 구조체의 장점: 
이들 변수를 동시에 메모리 저장공간을 할당받으며 프로그램 내에서 연관성을 유지하며 저장공간을 사용하기 위해서 사용된다.
